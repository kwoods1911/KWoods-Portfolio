Javascript is a client side scripting language, that is mainly use to add interactivity to static webpages. Jquery is a Javascript library used to make writing javascript more efficient. I have a firm understanding of Javascript and Jquery. Using both I add interactivity to my web sites. I strive to create a smooth experience for the user of all my websites.
